QHSE Management Platform — ISO Suite


Project Description:

QHSE Management Platform is an integrated management system dashboard designed around real-world ISO standards — ISO 9001, ISO 14001, ISO 45001, and ISO 27001.
The platform simulates a full enterprise-level QHSE system, enabling organizations to manage risks, incidents, audits, compliance, and environmental performance through a centralized, interactive dashboard.

Tech Stack / Built With:
- React
- TypeScript
- Vite
- Recharts
- Lucide React
- CSS Variables (custom theming system)


Features:

- Integrated dashboard covering all ISO standards
- Risk assessment module with interactive matrix and scoring system
- Incident tracking system with reporting and status updates
- Audit management for internal and external audits
- Document control with versioning and approval workflow
- Compliance checklist with ISO clause-level tracking
- Environmental performance tracking (emissions, waste, energy, water)
- Interactive charts and data visualizations
- Dark / light theme toggle (persisted in localStorage)
- Fully responsive layout (mobile, tablet, desktop)
- Collapsible sidebar navigation
- Modular and scalable component architecture
 - Modules:
Dashboard — KPI overview across all systems
Risk Assessment — hazard identification and evaluation
Incident Tracking — reporting and managing workplace incidents
Audit Management — audit scheduling and findings tracking
Document Control — document lifecycle and approvals
Compliance Checklist — ISO compliance monitoring
Environmental Tracker — sustainability and environmental metrics


Getting Started / Installation:

- Clone the repository:
git clone https://github.com/danemajstorcev/qhse-platform

- Navigate to the project folder:
cd qhse-platform

- Install dependencies:
npm install

- Run the development server:
npm run dev

- Open in browser:
http://localhost:5173

https://qhse-platform-chi.vercel.app/
